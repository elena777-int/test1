# Отчет по практической работе: Сетевое взаимодействие Docker и Ansible

## Цель работы
Настройка изолированной сети в Docker и проверка связи между контейнерами с помощью Ansible.

## Выполненные шаги
1. **Обновление системы**: Выполнено `dist-upgrade` на Alt Linux.
2. **Настройка Ansible**: Установлены коллекции `community.docker`.
3. **Развертывание**: Написан плейбук для создания сети `prod_network` и запуска двух контейнеров.

## Проверка связи
Для проверки сетевой связности использовалась команда:
`docker exec -it helper_prod ping game_prod`
